Vanligtvis faktuerar microsoft baserat på mängden data som skickas. Men eftersom vi är studenter så tillåts vi testa detta utan kostnad :)

Azure kontrollerar via github om det har blivit någon uppdatering och har det blivit det så triggas en ny build och publicering
